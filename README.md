# Jira Frontend

Jira-like project management frontend built with React and TypeScript.
Designed to work with an existing Spring Boot Jira backend.

---

## Features

- Authentication (JWT)
- Projects and Boards
- Issues (Create, Update, Delete)
- Status workflow
- Assignees
- Light / Dark mode
- Responsive UI

---

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- REST APIs

---

## Project Structure

src/
components/
pages/
services/
hooks/
styles/
App.tsx
main.tsx

---

## Getting Started

1. Clone the repository

git clone https://github.com/your-username/jira-frontend.git  
cd jira-frontend

2. Install dependencies

npm install

3. Run development server

npm run dev

App runs on http://localhost:5173

---

## Backend

Requires Spring Boot Jira backend to be running.
